# Cpufreq2

**程序简介**
一款无需 Root 或 ADB 权限的 Android CPU 监控工具。通过某些方案实现无`/proc/stat` 读取权限和无adb权限下的 CPU 使用率监控与可视化展示。

**功能**
1. 读取处理器占用率：通过分析 CPU 空闲时间与总运行时间的差值计算占用率。`/proc/stat` 不可用时，自动降级至读取time_in_state与空闲状态统计。
2. 实时悬浮窗：支持在屏幕任意角落以悬浮窗形式实时显示 CPU 使用率与频率。

**已知问题**
在 CPU 满载场景下，因空闲时间趋近于零，可能被误判为空载。
